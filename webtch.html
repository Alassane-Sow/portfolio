Portfolio Website Design

In Webtechnologie intro bouwde ik deze portfolio website! Het design proces begon met wireframes in Figma om de layout te plannen.

Design proces:

  • Wireframes maken in Figma
  • Color scheme bepalen
  • Typography selecteren
  • Responsive breakpoints plannen

Bootstrap 5 Framework

Bootstrap 5 was de basis voor deze website. Dit CSS framework biedt ready-made componenten en een krachtig grid systeem voor snelle, professionele ontwikkeling.

Bootstrap features:

  • Grid System: Flexibele 12-kolom layout voor responsive design
  • Components: Cards, navigation, buttons en meer
  • Utilities: Margin, padding, colors via utility classes
  • Mobile-first: Automatisch responsive op alle schermformaten
  • Customization: Eigen CSS voor persoonlijke styling

HTML5 & CSS3 Development

De website bouwen met clean, semantische HTML5 code en moderne CSS3 styling. Bootstrap 5 framework zorgde voor een responsive layout.

Technologieën:

  • HTML5 semantische elementen
  • CSS3 custom properties
  • Bootstrap 5 grid system
  • Custom CSS voor branding

Responsive Design & Deployment

De website werkt perfect op alle devices door mobile-first ontwikkeling. Deployment op Netlify maakt de site wereldwijd toegankelijk.

Resultaat:

  • 100% responsive op alle devices
  • Snelle laadtijden
  • SEO-vriendelijke structuur
  • Live op alassane-sow.netlify.app

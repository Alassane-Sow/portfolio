Serverbeheer Project

VMware & Virtualisatie

Voor we met fysieke servers werkten, leerden we virtualisatie met VMware Workstation. Hiermee konden we meerdere besturingssystemen draaien op één computer - ideaal voor testen en leren.

Wat ik leerde:

  • VMware Workstation installeren en configureren
  • Virtuele machines aanmaken en beheren
  • Virtual hardware configureren (RAM, CPU, disk)
  • Snapshots maken voor backup en testing
  • Netwerk configuratie tussen VM's

Windows 11 Virtuele Installatie

We installeerden Windows 11 als virtuele machine om te oefenen met moderne client-besturingssystemen zonder onze fysieke computers te beïnvloeden.

Installatie stappen:

  • Windows 11 ISO downloaden
  • VM aanmaken met TPM 2.0 en UEFI ondersteuning
  • Windows 11 installeren met juiste instellingen
  • VMware Tools installeren voor betere prestaties
  • Testen van Windows 11 features

Voordeel van virtualisatie: je kan experimenteren zonder risico voor je hoofdsysteem!

Windows Server Installatie

Bij Serverbeheer leerde ik werken met Windows Server. Van installatie tot configuratie - ik zette servers op en maakte ze klaar voor gebruik in een bedrijfsnetwerk.

Wat ik leerde:

  • Windows Server 2019/2022 installeren
  • Server rollen en features toevoegen
  • Initial configuration wizard
  • Network settings configureren

Server Configuratie & Services

Na de installatie kwam de configuratie. Services instellen, gebruikers beheren en security instellingen waren belangrijke taken.

Configuratie taken:

  • Active Directory Domain Services
  • DNS en DHCP services
  • File and Storage Services
  • Windows Updates en patches

Windows Admin Center

Windows Admin Center is een moderne web-based tool voor remote server management. Dit maakt centraal beheer van meerdere servers mogelijk.

Admin Center features:

  • Remote desktop connections
  • Performance monitoring
  • Event viewer en logs
  • PowerShell console toegang

Linux Command Line

Bij Besturingssystemen leerde ik intensief werken met de Linux terminal. De command line is essentieel voor efficiënt systeembeheer en automatisering.

CLI Vaardigheden

  • Navigatie: cd, ls, pwd, find
  • File management: cp, mv, rm, mkdir
  • Permissions: chmod, chown, umask
  • Text processing: cat, grep, less
Linux CLI Admin

Bash Scripting

Bash scripts schrijven voor automatisering van repetitieve taken. Dit spaart veel tijd en voorkomt menselijke fouten bij routine taken.

Scripting Skills

  • Variables en loops in bash
  • Conditional statements (if/else)
  • Functions voor herbruikbare code
  • Cron jobs voor scheduling
  • Error handling en debugging
Scripts Automation

Gebruikers & Groepen

Leerde werken met gebruikersbeheer in Linux: het aanmaken, wijzigen en verwijderen van gebruikers en groepen. Dit is een basisvaardigheid voor systeembeheer.

User Management

  • Aanmaken van gebruikers via useradd
  • Groepen beheren met groupadd
  • Gebruikers toevoegen aan groepen (usermod -aG)
  • Wachtwoorden instellen met passwd
  • Controleren met id en getent
Security Access Control

Bestandsrechten & Machtigingen

Leerde werken met Linux-bestandsrechten via chmod, chown en chgrp. Dit is essentieel voor toegangscontrole en security.

Permission Management

  • Wijzigen van bestandsrechten (chmod)
  • Eigenaar wijzigen (chown, chgrp)
  • Symbolische en numerieke modus (bijv. 755, 644)
  • Controleren van rechten met ls -l
  • Recursieve rechten voor directories
